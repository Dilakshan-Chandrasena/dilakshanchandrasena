Introduction to Bootstrap framework

2mins | 20.12.2021 | 02

Bootstrap is the most popular CSS framework for responsive and mobile-first websites. Before jumping into the Bootstrap framework, letâ€™s see what is a framework. Framework is like a place where we can use already existing codes to use in our projects. Existing codes in the sense is a set of codes that wrote by the other developers for a certain functionality of a software like login. Suppose we are building a software and we want add the login functionality. We can use a framework and grab the codes that already there and configure them as we want to build login functionality instead of coding it from the scratch. Basically, frameworks are there to make developers lives easy.

Bootstrap is developed by Mark Otto and Jacob Thornton at Twitter. First version was released on August 19th, 2011 licensed under MIT (Massachusetts Institute of Technology ). Bootstrap is a free tool which we can download from their official website and its open source. We can customize it as we want. Bootstrap is a front-end development framework which means it shall use for UI development. We can not use it in the server side /back-end. Bootstrap is used with HTML and CSS to make much more interactive UIs while putting a lesser effort. It will make developers lives easy.

Bootstrap provides pre-defined CSS which can use directly to style web pages. Also, bootstrap supports responsive web designs. Nowadays, people are using large scale of screen sizes in devices. Our web pages should be move with those screen sizes effectively in order to give a good experience for the users. Since Bootstrap provides different type of layouts, we can define different layouts for different screen sizes which allows us to make our web pages responsive.

Bootstrap has a god browser compatibility as well which will be an extra advantage.

HTML and CSS are the pre requisites for learning or using Bootstrap. Since Bootstrap comes with several JavaScript components in the form of jQuery plugins, it will be an added advantage if you know about JavaScript and jQuery. Each bootstrap components contains HTML structure, CSS styling and if necessary, JavaScript too.



How Bootstrap works?

As I mentioned earlier bootstrap has pre-defined HTML and CSS classes to use. Suppose, we have a button to design to use in our web page. We might want to change the font family, font color, size of the button, background, border and etc. Instead of writing codes to all those stuffs, simply we can use a Bootstrap class which is already there to use and design the same thing.
